Popover

Popover 구성 요소는 Tooltip(도구 설명)과 유사합니다. 사용자가 요소를 클릭할 때 나타나는 팝업 상자입니다.

차이점은 팝오버에 훨씬 더 많은 콘텐츠가 포함될 수 있다는 것입니다.

팝오버를 생성하려면 data-toggle="popover" 속성을 요소에 추가하십시오.

title 속성을 사용하여 팝오버의 헤더 텍스트를 지정하고 data-content 속성을 사용하여 팝오버 본문 내부에 표시되어야 하는 텍스트를 지정합니다.

주의: Popover는 jQuery로 초기화해야 합니다. 지정된 요소를 선택하고 popover() 메서드를 호출합니다.

Positioning Popovers

기본적으로 팝오버는 요소의 오른쪽에 나타납니다.

data-placement 속성을 사용하여 요소의 위쪽, 아래쪽, 왼쪽 또는 오른쪽에 도구 설명의 위치를 ​​설정합니다.
