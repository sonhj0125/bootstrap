Tooltip

tooltip(도구 설명)은 사용자가 요소 위로 마우스 포인터를 이동할 때 나타나는 작은 팝업 상자입니다.

툴팁을 생성하려면 data-toggle="tooltip" 속성을 요소에 추가하세요.

제목 속성을 사용하여 툴팁 내부에 표시되어야 하는 텍스트를 지정합니다.

주의: 도구 설명은 jQuery로 초기화해야 합니다. 지정된 요소를 선택하고 tooltip() 메서드를 호출합니다.

Positioning Tooltips

기본적으로 tooltip(도구 설명)은 요소 위에 표시됩니다.

data-placement 속성을 사용하여 요소의 위쪽, 아래쪽, 왼쪽 또는 오른쪽에 도구 설명의 위치를 ​​설정합니다.
